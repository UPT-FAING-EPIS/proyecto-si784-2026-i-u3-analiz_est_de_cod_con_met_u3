# Roadmap de Versiones

Este documento traza la evolución de la aplicación a lo largo de sus diferentes versiones.

## Versión 1.0 (MVP)
- [x] Configuración de flujos CI/CD con GitHub Actions.
- [x] Análisis estático de código (SonarCloud, Snyk, Semgrep).
- [x] Pruebas unitarias y de integración.
- [x] Pruebas de Interfaz con Playwright.
- [x] Pruebas de Mutación con Mutmut.

## Versión 1.1
- [ ] Integrar reportes detallados en un solo dashboard web (GitHub Pages).
- [ ] Ampliar cobertura de código a >80%.
- [ ] Añadir validación de infraestructura en Terraform con Checkov.

## Versión 2.0
- [ ] Implementación de nuevas funcionalidades en el backend.
- [ ] Migración completa de despliegue a entorno cloud usando los scripts de Terraform.
